# PORelDICE

This is the implementation for our paper **Relaxed Stationary Distribution Correction Estimation for Improved Offline Policy Optimization** in Jax.

This codebase is built upon [IQL](https://github.com/ikostrikov/implicit_q_learning.git) repository.

## Run Experiments 

Mujoco

    $ ./run_mujoco.sh

Antmaze

    $ ./run_antmaze.sh

Kitchen

    $ ./run_kitchen.sh
